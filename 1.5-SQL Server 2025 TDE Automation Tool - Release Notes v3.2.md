## ✨ What's New in v3.2

### 🎯 Major Enhancement: Idempotent Operations

**Idempotent = Safe to Run Multiple Times**

The tool now intelligently checks if objects already exist before creating them:

```
Before v3.2:
  ❌ Running Phase 2 twice would fail on second run
  ❌ Error: "Master Key already exists"
  ❌ No way to re-run phases safely

After v3.2:
  ✅ Phase 2 checks if Master Key exists
  ✅ If exists: Skips creation (no error)
  ✅ If not: Creates it
  ✅ Safe to run phases multiple times
```

## 📋 Implementation Details

### Master Key Detection
```sql
-- Check if Master Key exists
SELECT COUNT(*) FROM sys.symmetric_keys WHERE symmetric_key_id = 101

-- If result > 0: Master Key exists (skip)
-- If result = 0: Master Key doesn't exist (create)
```

### Certificate Detection
```sql
-- Check if Certificate exists
SELECT COUNT(*) FROM sys.certificates WHERE name = 'SonyDB_TDE_Cert'

-- If result > 0: Certificate exists (skip)
-- If result = 0: Certificate doesn't exist (create)
```

### DEK Detection
```sql
-- Check if DEK exists
SELECT COUNT(*) FROM sys.dm_database_encryption_keys 
WHERE database_id = DB_ID('SonyDB')

-- If result > 0: DEK exists (skip)
-- If result = 0: DEK doesn't exist (create)
```
