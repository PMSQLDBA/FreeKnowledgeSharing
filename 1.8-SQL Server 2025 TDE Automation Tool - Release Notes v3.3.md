## 🎉 Feature 2: Renew Certificate

### What It Does
```
Fully automated certificate renewal process:
  Step 1: Verify current certificate
  Step 2: Create new certificate
  Step 3: Backup new certificate
  Step 4: Switch DEK to new certificate
  Step 5: Verify new certificate protecting database
  Step 6: Display renewal summary
  
Total time: 15-30 minutes
Downtime: Minimal (15-30 minutes, new connections only)
```

### When to Use
```
Recommended: 7 days before expiry
Safe: 1-7 days before expiry
Latest: 1 day before expiry
NEVER: After expiry

For your database:
  First renewal: July 2027 (11 months from now)
  Then: Every year after that
```

### Step-by-Step Process

#### Step 1: Verify Current Certificate
```
Query: SELECT name, expiry_date, DATEDIFF(DAY, GETDATE(), expiry_date)
       FROM sys.certificates WHERE name = 'Karthikeya_TDE_Cert'

Output: Karthikeya_TDE_Cert | 2027-07-18 | 7 days remaining
Status: Ready to renew if days < 30
```

#### Step 2: Create New Certificate
```
Action: CREATE CERTIFICATE [Karthikeya_TDE_Cert_New]
        WITH SUBJECT = '...', EXPIRY_DATE = '2028-07-18'

Output: ✓ New certificate 'Karthikeya_TDE_Cert_New' created
Status: New certificate ready for use
```

#### Step 3: Backup New Certificate (CRITICAL)
```
Action: BACKUP CERTIFICATE [Karthikeya_TDE_Cert_New]
        TO FILE = '...Karthikeya_TDE_Cert_New.cer'
        WITH PRIVATE KEY (FILE = '...Karthikeya_TDE_Key_New.pvk', ...)

Output: ✓ New certificate backed up to:
          .cer: E:\path\TDE_Testing\Karthikeya_TDE_Cert_New.cer
          .pvk: E:\path\TDE_Testing\Karthikeya_TDE_Key_New.pvk
Status: Backup files secured
```

#### Step 4: Switch DEK to New Certificate
```
Action: ALTER DATABASE ENCRYPTION KEY
        ENCRYPTION BY SERVER CERTIFICATE [Karthikeya_TDE_Cert_New]

Output: ✓ DEK successfully switched to 'Karthikeya_TDE_Cert_New'
Status: Database now protected by new certificate
```

#### Step 5: Verify New Certificate
```
Query: SELECT c.name FROM sys.dm_database_encryption_keys dek
       LEFT JOIN sys.certificates c ON dek.encryptor_thumbprint = c.thumbprint
       WHERE database_id = DB_ID('Karthikeya')

Output: Karthikeya_TDE_Cert_New
Status: ✓ Verification successful
```

#### Step 6: Renewal Summary
```
════════════════════════════════════════════
CERTIFICATE RENEWAL SUMMARY
════════════════════════════════════════════
Database: Karthikeya
Old Certificate: Karthikeya_TDE_Cert
New Certificate: Karthikeya_TDE_Cert_New
Backup Files:
  - E:\path\TDE_Testing\Karthikeya_TDE_Cert_New.cer
  - E:\path\TDE_Testing\Karthikeya_TDE_Key_New.pvk
Status: RENEWED & ACTIVE
Next Renewal: 2028-07-18
════════════════════════════════════════════
```
